Webprogramozás I. Előadás beadandó
=====================================

Ez a repo a Neumann János Egyetem (NJE) **Webprogramozás I.** kurzusának előadás beadandó feladatához készült.

Áttekintés
==========

A feladat a leírás szerinti részfeladatok megvalósítása 2 fős csapatban.

Éles demo
=========

A projekt működés közben megtekinthető az alábbi linken:
https://webprog.biglan.net

Felhasznált technológiák és nyelvek
===================================

A fejlesztés során az alábbi nyelveket és technológiákat alkalmaztuk:

* HTML5
* CSS3
* JavaScript
* PHP
* React + Axios
* Bootstrap
